How to use the app

1) Open the folder in VS Code

2) Enter 'cd server' into the terminal followed by 'npm run dev'
    a) This will start the Node.js/Express Backend

3) Enter 'cd client' into the terminal followed by 'npm start'
    a) This will start the React Frontend and open a browser to the page.

4) You will be greeted by this page https://imgur.com/yJ7rwJZ

5) Navigate to the 'Load Current Bills' to display current data https://imgur.com/B9S3kgs

6) Fill in the field and add a bill https://imgur.com/VDEcl3e
